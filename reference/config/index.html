Configuration file

You configure the way Marcel builds your site using the marcel.config.js file. You only need this file if you want to diverge from the default configuration.

List of options

Base URL

  • base (default: /)

Folders

  • contentDir (default: content)
  • dataDir (default: data)
  • templateDir (default: templates)
  • staticDir (default: static)
  • distDir (default: dist)
  • filterDir (default: filters)

Templates

  • templateExt (default: html)

Lists

  • lists

Permalinks

  • permalinks.single (default here)
  • permalinks.list (default here)
